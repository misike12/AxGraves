## Minecraft 1.21.11 Compatibility Update

This version of AxGraves includes updates for Minecraft 1.21.11 compatibility, addressing NMS changes in the latest Minecraft versions.

### What's New
- ✅ Updated to support Minecraft 1.21.11 (Purpur build 2537-068b0d5)
- ✅ Enhanced error handling for NMS compatibility issues
- ✅ Improved logging for debugging
- ✅ Graceful degradation if visual elements fail

### For Server Administrators
If you're running Minecraft 1.21.11 or later, this update includes:
- Updated axapi dependency (1.6.0) with MC 1.21.x support
- Comprehensive error handling for initialization failures
- Better error messages for troubleshooting
